# pythonCourseFinalProject

##  Поставленная перед собой задача:

1) Сервис получения данных от удаленного стороннего сервиса на основе WSDL-описания

2) Запуск сервиса осуществляется раз в квартал

3) Доступ к удаленному сервису осуществляется по логину\паролю

4) Получение данных осуществляется частями, т.к. время соединения меньше требуемого для получения данных

5) Сохранение данных осуществляется в БД PostgreeSQL

6) Результат работы сервиса отправляется оператору на почту

7) Логирование процесса работы сервиса и возможность ручного запуска функционала получения данных
